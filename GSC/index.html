A series of examples involving a Girl Scout Cookie ordering application. "index.html"
  1. PHP is used to create the menu and display the default image
  2. PHP is used to synchronize the menu selection with the image
  3. PHP is used to deal with what is entered into the text box
  4. PHP is used to allow the use of the pulldown menu without placing an order
  5. JavaScript is used to do client-side error checking
